Python Resume Builder

A Python-based Resume Builder that generates professional PDF resumes using the ReportLab library.

Features

- Generate professional PDF resumes
- User-friendly resume format
- Customizable sections
- Automatic PDF generation
- Clean and structured layout

Requirements

- Python 3.x
- ReportLab

Installation

pip install reportlab

Run the Project

python resume_builder.py

Technologies Used

- Python
- ReportLab
